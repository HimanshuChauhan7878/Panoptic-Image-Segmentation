# 🖼️ Panoptic Image Segmentation

An AI-powered **Panoptic Image Segmentation** project that detects, classifies, and segments objects within an image with an accuracy of **up to 95%**. By combining semantic segmentation and instance segmentation, the model provides a complete scene understanding by identifying both individual object instances and background regions.

---

## 🎯 Objective

The goal of this project is to perform **Panoptic Segmentation**, enabling comprehensive scene analysis by assigning every pixel in an image to either a distinct object instance or a semantic class. This approach is widely used in autonomous driving, robotics, medical imaging, and smart surveillance systems.

---

## ✨ Features

- 🧠 AI-powered image segmentation
- 🎯 Detects and classifies multiple objects in a single image
- 🖼️ Pixel-level segmentation
- 📊 Up to **95% detection accuracy**
- ⚡ Fast and efficient inference
- 📷 Supports images of various resolutions
- 🎨 Generates segmented output with color-coded masks
